HTML (Hyper Text Markup Language)

HTML är ett, som många tror, programmeringsspråk. Men det är fel eftersom HTML är ett märkspråk, detta språket kommunicerar med webbläsaren så att den vet var informationen ska synas på webbsidan. HTML används till att koda webbsidornas skelett enkelt sagt. Med HTML kodar man dokumentets struktur (rubriker, stycken m.m.) och till en väldigt grundläggande nivå hur dokumentet ska se ut på front-end.

Från början var det bara möjligt att infoga bilder i koden, men sen några versioner tillbaka kan man infoga information i typ av datoprogram (java och javasvrips. Javaascript anvarar för funktionerna på webbsidan som byggs.) samt instruktioner för hur hemsidan skall se ut (CSS). Det är också möjligt att länka till andra webbsidor. Den senaste versionen är HTML5 som släpptes år 2014.

CSS (Cascading Style Sheet)

CSS är ett språk som sammarberat med HTML. Med CSS sköter man hela stilen och utseendet på hemsidan. Alltså det man har kodat i HTML kan man sedan ändra utseendet på i CSS, ex. färger, storlek, position, ändra fonten på texten, tjockleken och storleken på texten och mycket mycket mera.
