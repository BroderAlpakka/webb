HTTP (HyperText Transfer Protocol)

HTTP är ett protokoll som används för att skicka ut en hemsida till World Wide Web (W3C). Det blir mycket vanligare idag att HTTP byts ut mot HTTPS som är en säkrare protokoll och använder sig av SSL (Secure Socket Layer) för att skicka och ta emot käsnlig information till och från webbservern säkrare och krypterad. Fördelen med just HTTPS är att känslig information såsom kreditkorts information eller annat inte hamnar i någon annans händer. För att en sida ska vara HTTPS så måste ett SSL certifikat vara installerad på servern. Detta certifikatet fungerar som en legitimation och försäkrar klienter om att webbservern faktiskt är en webbserver.

Webbläsare (Chrome, Fireforx, Opera, Explorer etc.)

Webbläsare är kort sagt ett program som hämtar och tolkar webbsidor som är lagda på en webbserver på internet.
Den viktiga delen som gör att man kan komma åt hemsidor är tjänsten World Wide Web som skapades av Timothy Berners-Lee år 1990. Timothy skrev också klienten för Nextstep som var första webbläsaren som gjorde det möjligt att surfa mellan olika webbsidor som då fanns. Ett par år senare lanserades webbläsaren Mosaic, eftersom den var enklare att använda så började fler folk använda den. Sedan lanserades webbläsaren Netscape som dominerade ett par år tills Microsoft Internet Explorer började ta över trafiken, och år 2002 tog de över hela 96% av trafiken. Detta gjorde att Netscape idag bara finns kvar till namnet.

Senare under 2000-talet blev det ett nytt webbläsarkrig mellan Internet Explorer och Mozilla Fireforx, google Chrome samt Safari och Opera. Dessa fem tog över makten i viss del och är idag dem största webbläsarna. Fast Internet Explorer fortfarande används idag så är det sällan man träffar folk som föredrar Explorer före ex. Google Chrome eller Mozilla Firefox. Enligt statistiken från W3Schools från år 2016 så låg Internet Explorer på 3 plats med 5.2% använding, strac efter Mozilla Firefox (17.1%) och första plats Google Chrome (71.9%). Enligt statistiken från Net Applications från samma år kan man se att Internet Explorer ligger på 2 plats med 29.6% strax efter Google Chrome som la på 50.95%.
